## 项目概述
这是一个基于Servlet+JSP技术开发的在线购物网站，包含顾客和销售管理两大功能模块。

## 技术栈
- 前端：HTML + CSS + JavaScript + JSP
- 后端：Servlet (Jakarta EE 6.0)
- 数据库：MySQL
- 服务器：Tomcat 11

## 项目结构

```
├── src/                          # Java源代码目录
│   ├── db.properties             # 数据库配置文件
│   ├── RemoveBOM.java            # BOM字符处理工具类
│   └── com/                      # 主包
│       └── shopping/             # 项目包
│           ├── controller/       # Servlet控制器
│           ├── dao/              # 数据访问对象
│           ├── model/            # JavaBean模型
│           ├── service/          # 业务逻辑层
│           └── util/             # 工具类
├── WebContent/                   # Web资源目录
│   ├── css/                      # CSS样式文件
│   ├── images/                   # 图片资源
│   ├── js/                       # JavaScript文件
│   ├── jsp/                      # JSP页面
│   │   └── admin/                # 管理员页面
│   └── WEB-INF/                  # WEB配置目录
│       ├── lib/                  # 第三方库
│       └── web.xml               # Web应用配置文件
├── shoppingdb.sql                # 数据库脚本
└── README.md                     # 项目说明文档
```

## 功能模块

### 一、顾客功能
1. **注册、登录、注销**
2. **产品列表展示与搜索**
3. **产品详情查看**
4. **购物车管理**
5. **订单生成与支付**
6. **订单状态与历史查看**
7. **用户个人信息管理**

### 二、销售管理
1. **商品目录管理**
   - 添加商品
   - 修改商品
   - 删除商品
2. **订单管理**
   - 查看所有订单
   - 更新订单状态
3. **销售统计报表**

## 数据库设计

### 主要表结构
- `users`：用户信息表
- `products`：商品信息表
- `orders`：订单表
- `order_items`：订单项表
- `shopping_cart`：购物车表

## 部署说明

### 1. 数据库部署
- 创建MySQL数据库
- 执行`shoppingdb.sql`脚本创建表结构和测试数据
- 修改`src/db.properties`文件，配置数据库连接信息（URL、用户名、密码）

### 2. 项目部署
- **推荐使用Tomcat 11服务器**
- 将项目打包为WAR文件或直接部署到Tomcat的webapps目录
- 启动Tomcat服务器
- 在浏览器中访问`http://8.148.155.248:8080/shopping`（云服务器部署）或`http://localhost:8080/shopping`（本地部署）
