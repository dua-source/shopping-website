# 导出本地数据库到SQL文件

## 执行步骤

1. **编译ExportDatabase.java**
   - 使用javac命令编译程序，确保包含MySQL驱动依赖

2. **运行程序导出数据库**
   - 执行编译后的程序，连接本地数据库
   - 生成完整的shoppingdb.sql文件，包含：
     - 数据库创建语句
     - 所有表结构
     - 所有表数据
     - 正确的字符集设置

3. **验证导出结果**
   - 检查生成的shoppingdb.sql文件
   - 确保包含所有表和数据
   - 确保字符集设置正确

4. **使用SQL文件部署到云服务器**
   - 将生成的shoppingdb.sql文件上传到云服务器
   - 在云服务器上执行该SQL文件
   - 确保云服务器数据库与本地完全一致

## 命令执行顺序

1. **编译程序**：
   ```powershell
   javac -cp "WebContent\WEB-INF\lib\mysql-connector-j-9.4.0.jar" ExportDatabase.java
   ```

2. **运行程序**：
   ```powershell
   java -cp ".;WebContent\WEB-INF\lib\mysql-connector-j-9.4.0.jar" ExportDatabase
   ```

3. **验证结果**：
   ```powershell
   Get-Content shoppingdb.sql | Select-Object -First 50
   ```

## 预期结果
- 生成完整的shoppingdb.sql文件
- 文件包含所有表结构和数据
- 云服务器上的数据库与本地完全一致
