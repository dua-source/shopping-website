# 编译所有Java文件并重新打包WAR文件计划

## 步骤1：编译所有Java文件
使用javac命令编译所有Java文件，指定正确的类路径和编码：
```bash
javac -encoding UTF-8 -d WebContent/WEB-INF/classes -cp "WebContent/WEB-INF/lib/*" src/com/shopping/**/*.java
```

## 步骤2：检查编译结果
- 如果编译成功，继续下一步
- 如果编译失败，检查错误信息并修复问题

## 步骤3：重新打包WAR文件
使用jar命令重新打包WAR文件：
```bash
jar -cvf shopping-new.war -C WebContent .
```

## 步骤4：验证WAR文件
检查生成的WAR文件是否包含所有必要的文件和目录结构。
