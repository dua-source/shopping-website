# 修改db.properties指向阿里云RDS数据库

## 1. 找到db.properties文件

db.properties文件通常位于项目的src目录下，具体路径为：
`src/com/shopping/util/db.properties`

## 2. 修改连接配置

将文件内容修改为指向阿里云RDS数据库，使用以下连接信息：
- 实例地址：rm-7xvtn0z98iy86ses1.mysql.rds.aliyuncs.com:3306
- 数据库账号：administrator
- 数据库名称：shoppingdb

## 3. 配置内容模板

```properties
# 阿里云RDS数据库连接配置
db.url=jdbc:mysql://rm-7xvtn0z98iy86ses1.mysql.rds.aliyuncs.com:3306/shoppingdb?useSSL=false&serverTimezone=UTC&characterEncoding=utf8
db.username=administrator
db.password=您的数据库密码
db.driver=com.mysql.cj.jdbc.Driver
```

## 4. 保存配置文件

修改完成后，保存db.properties文件。

## 5. 测试连接

修改配置后，启动Tomcat服务器，部署项目，访问网站首页，测试数据库连接是否成功。

## 注意事项

1. 确保数据库密码正确，这是您在创建RDS实例时设置的密码
2. 确保阿里云RDS实例的安全组和白名单设置正确
3. 确保项目中包含MySQL JDBC驱动（mysql-connector-java.jar）
4. 如果遇到连接问题，检查Tomcat日志文件获取详细错误信息

通过以上步骤，您应该能够成功将本地项目连接到阿里云RDS数据库。
