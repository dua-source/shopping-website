# 编译项目计划

## 任务目标
重新编译项目，生成最新的class文件

## 执行步骤
1. 清理旧的编译结果
2. 重新编译model层（先编译OrderItem.java，再编译其他model类）
3. 编译dao层
4. 编译util层
5. 编译service层
6. 编译controller层
7. 检查编译结果

## 预期结果
- 成功编译所有Java文件
- 生成最新的class文件
- 无编译错误
- 生成完整的build目录

## 执行命令
```powershell
# 清理旧结果
Remove-Item -Path build -Recurse -Force

# 创建build目录
New-Item -Path build -ItemType Directory -Force | Out-Null

# 编译model层
javac -d build -cp "WebContent\WEB-INF\lib\*" src\com\shopping\model\OrderItem.java
Get-ChildItem -Path src\com\shopping\model -Filter *.java -Recurse | Where-Object { $_.Name -ne "OrderItem.java" } | ForEach-Object { javac -d build -cp "build;WebContent\WEB-INF\lib\*" $_.FullName }

# 编译dao层
Get-ChildItem -Path src\com\shopping\dao -Filter *.java -Recurse | ForEach-Object { javac -d build -cp "build;WebContent\WEB-INF\lib\*" $_.FullName }

# 编译util层
Get-ChildItem -Path src\com\shopping\util -Filter *.java -Recurse | ForEach-Object { javac -d build -cp "build;WebContent\WEB-INF\lib\*" $_.FullName }

# 编译service层
Get-ChildItem -Path src\com\shopping\service -Filter *.java -Recurse | ForEach-Object { javac -d build -cp "build;WebContent\WEB-INF\lib\*" $_.FullName }

# 编译controller层
Get-ChildItem -Path src\com\shopping\controller -Filter *.java -Recurse | ForEach-Object { javac -d build -cp "build;WebContent\WEB-INF\lib\*" $_.FullName }

# 检查编译结果
Write-Output "编译完成，生成的class文件数量: $((Get-ChildItem -Path build -Filter *.class -Recurse).Count)"
```
